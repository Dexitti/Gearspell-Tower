Для игрока, когда шестеренок хватает на самый дешевый upgrade, над башней появляется "стрелочка" (обрабатывает HUD) и игрок может нажать на нее/'E' чтобы открыть панель (окно) улучшений. Игра в этот момент ставится на паузу. Панель улучшений представляет собой окно поверх игры (изначально есть на сцене, просто выключено), содержащее: - кнопку назад/'Esc' - кнопку открыть слот (должна быть иконка и ценна) сверху - пул из 3-х карточек на выбор в центре (над ценной подумаю), которые могут быть: --- новым снаряжением, если еще есть открытые и не экипированные, а также есть свободный слот (может быть открыт в этот же момент в *магазине) --- улучшением экипированного (правила приводил выше) - ниже три широкие кнопки глобальных усилений (хил, временный +реген, раскидать *мины по полю) подумай над их набором, а то 2 хила по сути не есть хорошо - где-то расположить текущие шестеренки, вокруг панели либо затемнить, либо сделать ее во весь экран с задником то есть это похоже на магазин/верстак, в котором игрок может потратить заработанные шестеренки подумай, как расположить элементы на панели

UpgradeData - хранит данные улучшения для отображаемой карточки
cost пока в них, но потом будет глобальное хранилище ценообразования
id нужен для сопоставления upgrade'а в выборе (UI) и upgrade'а в снаряжении (логика)
UpgradeStages должна ассоциироваться с int и определять уровень доступа снаряжения:
-1 допустим снаряжение не доступно (можно обрабатывать как отдельное поле)
0 доступно только снаряжение
1 (с учетом проверки наличия экипированного снаряжения), доступны обычные upgrade'ы
2 (если есть любой обычный, в пул добавляются улучшения-развилки. Особенность их в том, что при выборе одного, другие блокируются. Обычные остаются)
3 При подборе развилки, прочие развилки исключаются, остаются обычные и добавляется улучшение-активка

SaveManager - глобальный прогресс. Хранит открытое снаряжение и настройки игры.
- По сути массив строк или чисел, без разницы, как удобно
ProgressManager - хранит состояние сессии:
- текущая волна
- количество шестеренок
- количество хп башни переделай
- разблокированные слоты и снаряжение в них
- выбранные улучшения (int'ы или id, как удобно)
