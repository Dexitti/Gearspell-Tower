С убитых врагов выпадает ресурс - шестеренки. Игрок их может потратить:
- на слоты со снаряжением (до 3-х и 1 в рюкзаке (ref. Dead Cells))
- на прокачку текущего снаряжения
- на глобальные способности:
	- восстановить x% XP (~={blue}Восстановление=~)
	- временно улучшить пассивную регенерацию (~={green}Защита=~)
	- раскидать ловушки на поле (~={red}Вооружение=~)

Во время прокачки игроку предлагается выбор из трех улучшений.
У каждого ~={orange}снаряжения=~ есть несколько вариантов улучшения характеристик (механик) и ~={pink}активная способность=~. [[Снаряжение.png]] (структура - лес)
\*С боссов/после определенных волн игрок может выбрать билдообразующий артефакт(-ы)

Снаряжения хранятся в слотах (их число ограничено, но может быть увеличено с прокачкой) Изначально доступна часть снаряжений.
Игроку предлагается выбор обычных улучшений имеющегося снаряжения.
Если доступен пустой слот и есть еще открытые и не экипированные снаряжения, они могут попасться.
При наличии любого обычного улучшения, становятся доступны развилки для этого снаряжения.
Если есть развилка, то иное блокируется и становится доступна активка.

Edit:
Суть такая: Для игрока, когда шестеренок хватает на самый дешевый upgrade, над башней появляется "стрелочка" (я уже описывал, это обрабатывает HUD) и игрок может нажать на нее/'E' чтобы открыть панель (окно) улучшений. Игра в этот момент ставится на паузу Панель улучшений представляет собой окно поверх игры (изначально есть на сцене, просто выключено), содержащее: - кнопку назад/'Esc' - кнопку открыть слот (должна быть иконка и ценна) сверху - пул из 3-х карточек на выбор в центре (над ценной подумаю), которые могут быть: --- новым снаряжением, если еще есть открытые и не экипированные, а также есть свободный слот (может быть открыт в этот же момент в *магазине) --- улучшением экипированного (правила приводил выше) - ниже три широкие кнопки глобальных усилений (хил, временный +реген, раскидать *мины по полю) подумай над их набором, а то 2 хила по сути не есть хорошо - где-то расположить текущие шестеренки, вокруг панели либо затемнить, либо сделать ее во весь экран с задником то есть это похоже на магазин/верстак, в котором игрок может потратить заработанные шестеренки подумай, как расположить элементы на панели

UpgradeData - хранит данные улучшения для отображаемой карточки
cost пока в них, но потом будет глобальное хранилище ценообразования
id нужен для сопоставления upgrade'а в выборе (UI) и upgrade'а в снаряжении (логика)
UpgradeStages должна ассоциироваться с int и определять уровень доступа снаряжения:
-1 допустим снаряжение не доступно (можно обрабатывать как отдельное поле)
0 доступно только снаряжение
1 (с учетом проверки наличия экипированного снаряжения), доступны обычные upgrade'ы
2 (если есть любой обычный, в пул добавляются улучшения-развилки. Особенность их в том, что при выборе одного, другие блокируются. Обычные остаются)
3 При подборе развилки, прочие развилки исключаются, остаются обычные и добавляется улучшение-активка

SaveManager - глобальный прогресс. Хранит открытое снаряжение и настройки игры.
- По сути массив строк или чисел, без разницы, как удобно
ProgressManager - хранит состояние сессии:
- текущая волна
- количество шестеренок
- количество хп башни переделай
- разблокированные слоты и снаряжение в них
- выбранные улучшения (int'ы или id, как удобно)
